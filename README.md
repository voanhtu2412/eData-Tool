# eData-Tool (TaxCodeCollector)

Công cụ tự động thu thập thông tin mã số thuế doanh nghiệp (Tax Code Collector) sử dụng C# WPF và Microsoft Playwright với cơ chế ẩn danh nâng cao (kết nối trình duyệt thực qua CDP).

## Tính năng chính
- Tự động điều khiển trình duyệt để tìm kiếm và thu thập thông tin doanh nghiệp.
- Cơ chế chống phát hiện bot và vượt WAF/Cloudflare bằng cách tích hợp trực tiếp với Chrome thật qua cổng CDP (Chrome DevTools Protocol).
- Giao diện WPF trực quan, cấu hình động.
